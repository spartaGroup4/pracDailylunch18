추천하고싶은 점심 메뉴를 이미지와 추천 이유를 업로드하여  
점심 메뉴 고민시간을 줄여줄 수 있는 웹사이트 입니다.

---

### 목차

-   사용 기술
-   팀원 목록
-   프로젝트 소개
    -   와이어 프레임
    -   프로젝트 결과

### 사용 기술
Back-end
```
* Python 3
* Flask 2.0.1
* MongoDB 4.4
```
Front-end
```
* CSS
```
deploy
```
* AWS EC2 (Ubuntu 18.04 LTS)
```
